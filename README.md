# zintensity.dev

This is just my personal website, nothing special.

## Running

You would run this the same way you run any other Next.js app:

```console
$ git clone https://github.com/ZeroIntensity/zintensity.dev && cd zintensity.dev
$ npm run dev
# or
$ yarn dev
# or
$ pnpm dev
# or
$ bun dev
```
